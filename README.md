## ✨ Основные возможности

### 🌐 Веб-платформа (Admin & Staff Panel)
*   **Регистрация сотрудников:** Управление базой данных пользователей через личный кабинет администратора.
*   **Генерация QR-пропусков:** Мгновенное создание временных пропусков с привязкой к сессии.
*   **Встроенный сканер:** Режим охраны для считывания QR-кодов прямо через браузер (требует HTTPS).
*   **Корпоративный стиль:** Полное соответствие брендбуку «Ростелеком» (Inter font, RTK Purple/Orange).

### 📱 Мобильное приложение (Employee Pass)
*   **Native Security (Anti-Screenshot):** Реализована блокировка скриншотов и записи экрана на нативном уровне Android (Kotlin `FLAG_SECURE`). Пропуск невозможно скопировать или передать через снимок экрана.
*   **Динамический QR:** Код обновляется и подписывается HMAC-SHA256 хешем, предотвращая подделку.
*   **Таймер жизни:** Пропуск аннулируется автоматически через 5 минут после генерации.
*   **Симуляция СКУД:** Встроенный режим «Турникет» для проверки валидности кодов охраной прямо в приложении.

## 🛡️ Безопасность данных

1.  **Защита от подделки:** Каждый QR-код содержит в себе `username | timestamp | signature`. Подпись генерируется на сервере с использованием секретного ключа `RTK_INTERNAL_KEY_2026`.
2.  **Защита от утечек (Native Layer):** В мобильной версии нативно запрещен захват экрана. В веб-версии реализовано аннулирование сессии при потере фокуса или закрытии вкладки.
3.  **Хеширование паролей:** Все учетные данные защищены, сессии шифруются.
