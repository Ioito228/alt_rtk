# 🛡️ QR-Pass Scanner (Rostelecom Style)

Система цифровых пропусков с генерацией динамических QR-кодов и встроенным сканером для СКУД. Разработано с использованием корпоративного стиля «Ростелеком».

## ✨ Основные функции
- **Панель администратора**: регистрация новых сотрудников и управление базой.
- **Генерация QR-кодов**: создание временных пропусков (срок жизни 5 минут).
- **QR-Сканер**: встроенный в браузер сканер для мобильных устройств (требует HTTPS).
- **Защита данных**: автоматическое аннулирование пропуска при выходе из вкладки или потере фокуса.
- **Безопасность**: хеширование паролей (bcrypt) и авторизация на основе сессий.

## 🛠 Стек технологий
- **Backend**: Python 3.12, FastAPI, SQLAlchemy, Uvicorn.
- **Frontend**: HTML5, CSS3 (Inter font), JavaScript (Vanilla).
- **База данных**: SQLite.
- **QR**: `html5-qrcode` (JS), `qrcode` (Python).
- **Развертывание**: Yandex Cloud, Cloudflare Tunnel (HTTPS).
