# 🛡️ RTK Access: Точка входа

Комплексная система цифровых пропусков «Ростелеком» для СКУД. Включает в себя масштабируемую веб-платформу и защищенное мобильное приложение для сотрудников.

---

## 🌐 1. Веб-платформа (Web Interface)
Система управления доступом и регистрации через браузер.

### ✨ Основные функции
*   **Панель администратора:** Регистрация новых сотрудников и управление базой (SQLite).
*   **Генерация QR:** Создание временных пропусков (срок жизни 5 минут).
*   **QR-Сканер:** Встроенный в браузер сканер для КПП (требует HTTPS).
*   **Безопасность:** Автоматическое аннулирование пропуска при потере фокуса или закрытии вкладки.

### 🛠 Стек технологий
*   **Backend:** Python 3.12, FastAPI, SQLAlchemy.
*   **Frontend:** HTML5, CSS3 (Inter font), Vanilla JS.
*   **Deployment:** Yandex Cloud.

---

## 📱 2. Мобильное приложение (Android APK)
Нативное приложение для сотрудников на базе **Flet (Flutter + Python)**.

### ✨ Эксклюзивные функции мобильной версии:
*   **Защита от скриншотов (Anti-Screenshot):** Реализована через нативный слой Android (Kotlin/FLAG_SECURE). Приложение блокирует попытки сделать снимок экрана или запись видео с QR-кодом.
*   **Динамическая подпись:** Генерация подписи на основе HMAC-SHA256 для защиты от подделки кодов.
*   **Offline-ready UI:** Корпоративный интерфейс в стиле RTK Purple/Orange.

### 🛡 Безопасность (Native Layer)
В мобильной версии реализован класс `MainActivity.kt`, который взаимодействует с Window Manager Android для предотвращения утечки учетных данных сотрудников.

---

## 🚀 Как запустить / Собрать

### Веб-версия:
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите сервер: `uvicorn main:app --reload`

### Мобильная версия (Получение APK):
1. Перейдите во вкладку **Actions** в этом репозитории.
2. Выберите последний успешный билд **"Build Mobile APK"**.
3. В разделе **Artifacts** скачайте архив `rtk-mobile-apk`.
4. Распакуйте и установите `.apk` на Android-устройство.
